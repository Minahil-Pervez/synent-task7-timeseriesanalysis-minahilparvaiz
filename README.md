# Time Series Analysis and Stock Forecasting

## Overview
This project analyzes stock market data and predicts future stock prices using time series forecasting techniques.
The dashboard helps users understand historical stock performance and visualize future trends through forecasting models.

## Problem Statement
Stock market data changes over time and contains valuable trends. The objective of this project is to analyze historical stock prices, identify trends, and forecast future values using machine learning techniques.

## Tools and Technologies
Python
Streamlit
Pandas
Plotly
Prophet
Scikit Learn

## Project Workflow

1. Upload stock dataset
2. Data preprocessing
3. Date conversion and sorting
4. Trend analysis
5. Candlestick visualization
6. Moving average calculation
7. Forecast model training
8. Future prediction
9. Model evaluation

## Features
Stock dataset upload
Candlestick chart
Closing price trend analysis
Moving average analysis
Forecasting with Prophet
Forecast visualization
Trend component analysis
MAE calculation
RMSE calculation
Forecast data table

## Results

The dashboard successfully identifies historical trends and generates future stock price forecasts. Users can visualize market movements and evaluate forecasting performance using error metrics.

## Dataset

Stock price dataset containing Date, Open, High, Low, and Close columns.
